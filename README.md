Overview
The Nusrat Hospital website aims to provide patients and visitors with easy access to hospital services and resources. The website is user-friendly, mobile-responsive, and visually appealing, highlighting the hospital's facilities, medical staff, and departments.


Features
- Responsive Design: Works seamlessly across desktops, tablets, and mobile devices.
- Hero Section: Welcome message with hospital highlights.
- About Us: Detailed information about the hospital’s mission, vision, and values.
- Departments: Showcase of specialized departments with descriptions and images.
- Doctors: Profiles of doctors with expertise and contact links.
- Appointment Form: Users can book appointments online.
- FAQ Section: Answers to common patient questions.
- Gallery: Highlights hospital facilities, staff, and community events.
- Stats Section: Displays hospital statistics like number of doctors, labs, and awards.
- Social Links: Connect with the hospital on social media platforms.
- Contact & Location : Embedded Google Maps for directions and contact information.

Technologies Used
- **HTML5 & CSS3** for structure and styling
- **Bootstrap 5** for responsive design
- **JavaScript** for interactivity
- **AOS (Animate On Scroll)** for scroll animations
- **GLightbox** for media popups
- **Swiper.js** for sliders and galleries
- **FontAwesome & Bootstrap Icons** for icons

Installation

1- Clone the repository:
   git clone https://github.com/yourusername/nusrat-hospital.git

2- Navigate to the project folder:
cd nusrat-hospital

3- Open index.html in a web browser to view the website locally.

Usage

Use the navigation menu to explore different sections like About, Services, Departments, Doctors, and FAQ.

Submit appointment requests using the make appointment button.

View hospital facilities and staff through the gallery and department sections.
